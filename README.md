# Talentsy Sales Agent Bot

AI-агент продавец для Telegram. Ведёт диалог с лидами по методологии Talentsy.

---

## Быстрый старт

### 1. Получи токены

**Telegram Bot Token:**
- Открой @BotFather в Telegram
- Напиши `/newbot`
- Дай боту имя и username
- Скопируй токен

**OpenAI API Key:**
- Зайди на platform.openai.com
- Settings → API Keys → Create new secret key
- Скопируй ключ (показывается один раз)

---

### 2. Настрой доступ к Google Docs

Все 7 документов должны быть открыты по ссылке:
- Открой каждый документ
- Настройки доступа → Все у кого есть ссылка → Читатель
- Сохрани

---

### 3. Деплой на Railway (бесплатно)

1. Зайди на **railway.app** → Sign Up with GitHub
2. New Project → Deploy from GitHub repo
3. Загрузи эту папку в GitHub (или используй Railway CLI)
4. В настройках проекта добавь переменные окружения:
   - `TELEGRAM_TOKEN` = твой токен
   - `OPENAI_API_KEY` = твой ключ
5. Deploy

---

### 4. Локальный запуск (для теста)

```bash
# Установи зависимости
pip install -r requirements.txt

# Создай .env файл
cp .env.example .env
# Открой .env и вставь токены

# Запусти
python bot.py
```

---

## Команды бота

- `/start` — начать новый диалог (агент отправляет первое сообщение)
- `/reset` — сбросить историю диалога

---

## Как работает

1. Клиент пишет `/start` — агент отправляет приветствие с упоминанием диагностики
2. База знаний загружается из Google Docs автоматически при первом запросе
3. GPT-4o ведёт диалог по промту v3 с полной методологией Talentsy
4. История диалога хранится в памяти (сбрасывается при перезапуске)

---

## Обновление базы знаний

База знаний кэшируется при первом запуске. Чтобы обновить после изменений в Google Docs — перезапусти бота.
